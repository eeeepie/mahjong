# 台州麻将原型规则

这个文件描述当前 CLI 原型采用的规则和明确做过的简化。

## 当前规则

- 四人麻将，支持连打多局。
- 使用 136 张常规麻将牌，不包含花牌。
- 白板 `Ba` 作为万能牌处理，可以替代任意牌参与胡牌判断。
- 白板不参与吃、碰、杠副露，只在胡牌判断里当万能牌使用。
- 支持摸牌、打牌、吃、碰、杠、点炮胡、自摸胡。
- 胡牌按标准结构判断：`4 组面子 + 1 对将`，并会把已有副露一起算入。
- 玩家入桌默认 `100 point`。
- CLI 提供 `zh/ch` 与 `en` 两套语言界面，默认中文。
- 牌桌会显示“已见牌情报”，统计所有打出牌和明副露。
- 庄家胡牌连庄；闲家胡牌或流局，下家接庄。
- Bot 名字使用中二风格名字池。
- 玩家在出现听牌出路后，可以输入 `auto` / `托管` 进入自动托管，后续会自动胡牌并自动过非胡操作。

## 当前胡数与台数原型

### 胡数

- `10` 底胡起算。
- 明刻 `2-8` 记 `2 胡`，暗刻 `2-8` 记 `4 胡`。
- 明刻 `1/9/字牌` 记 `4 胡`，暗刻 `1/9/字牌` 记 `8 胡`。
- 明杠 `2-8` 记 `8 胡`，暗杠 `2-8` 记 `16 胡`。
- 明杠 `1/9/字牌` 记 `16 胡`，暗杠 `1/9/字牌` 记 `32 胡`。
- 门风将、`自摸`、`碰碰胡`、`嵌张`、`边张` 会额外加胡。

### 台数

- 门风刻/杠 `1 台`。
- 红中刻/杠、发财刻/杠各 `1 台`。
- `混一色 2 台`。
- `无白板 1 台`。
- `杠开 1 台`。
- `清一色`、`字一色`、`四风齐聚` 按封顶大牌处理。

## 当前结算原型

- 本局基础值按 `(10底胡 + 附加胡) x 2^台数` 计算，封顶 `100`。
- 点炮胡：只有放铳者支付整份。
- 自摸胡：其余三家都支付整份。
- 当前结算仍是便于 CLI 游玩的近似模型，不是完整线下台州地方结算表。

## 当前简化

- 还没有买马、包牌、抢杠胡、财神归位等完整台州地方细则。
- 还没有七对、十三幺等特殊胡型，也没有把所有大牌都做成完整地方判例。
- Bot 使用启发式出牌和副露选择，不是竞技级 AI。
- 大厅是本地模拟，房间和玩家都由脚本生成。

## 扩展建议

- 如果后续要补番种或地方规则，先在这里增加说明，再改 `scripts/mahjong_cli.py`。
- 如果以后加入更多地区玩法，建议把本文件拆成多个 `references/*.md`，让 `SKILL.md` 负责导航。
